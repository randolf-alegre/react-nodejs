# node-nextjs

Tech stack tools:
* NodeJS
* Prisma (ORM)
* NextJS
* ExpressJS
* Tailwind CSS
* Redux Toolkit
  
HOW TO:

Setup backend:
1. Go to root folder.
2. Run npm install
3. Run the command: npx prisma migrate dev --name init
4. Seed data: npx prisma db seed         
5. Run npm run start

Setup Frontend:
1.  Go to root folder.
2.  Run npm install
3.  Run npm run dev
